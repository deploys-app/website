From zero to live

One command
to production.

Point the CLI at a project and location, and Deploys rolls it out — image pull, replicas, routing, and a public HTTPS URL.

deploys cli 
# authenticate once with a service-account key
$ export DEPLOYS_AUTH_USER=ci@acme.iam.deploys.app
$ export DEPLOYS_AUTH_PASS=$DEPLOYS_KEY

# build, push, and roll out a web service
$ deploys deployment deploy \
    --project acme --location gke.cluster-rcf2 \
    --name web --image nginx:1.27 --port 80

✓ live at https://web.acme.rcf2.deploys.app
